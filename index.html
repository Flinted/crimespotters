Welcome to the CrimeSpotter

Selecting an area:

You can check a mile radius anywhere in England,
Wales or Northern Ireland by clicking on the Map.
Or enter any UK latitude and longitude.

Dates:

The drop down to the left allows you to select a month to return crime data for. If you change this it will return data for the currently selected location and any future selections.

Filtering Crimes:

Just click the category names in the info window to
just display this kind of crime.

Markers:

You can click on markers for more information
on the crime that occured there.
